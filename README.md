# Namma Studio

A modern and immersive creative agency website built with **HTML, CSS, and JavaScript**, featuring premium animations, interactive cursor effects, portfolio showcases, smooth transitions, and engaging user experiences inspired by modern digital design studios.

## ✨ Features

- Modern creative agency landing page
- Interactive custom cursor animations
- Dynamic portfolio showcase section
- Hover-based image reveal effects
- Smooth GSAP-inspired UI interactions
- Animated navigation overlay menu
- Video-enhanced user interactions
- Live digital clock display
- Service showcase section
- Premium typography and branding aesthetics
- Dark and modern visual design

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Shankar-soni-2006/Namma-Studio.git
```

### Navigate to the Project

```bash
cd Namma-Studio
```

### Run Locally

Simply open:

```text
index.html
```

in your browser.

Or use VS Code Live Server:

```bash
Right Click → Open with Live Server
```
